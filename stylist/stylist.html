Save a local file

Illustrations are stored as ZIP archives, with all inputs and data included. This is useful for sharing via email, though you might need to remove the .zip file extension to pass spam filters. An illustration can be saved as a ZIP archive in most modern browsers. This browser does not provide the needed access to the local filesystem.
